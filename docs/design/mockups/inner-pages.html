LOOP · lessons

The toolbox learns

captured at deliver / debug→ .marvin/memory/→ recalled at the next task-start↺

Lessons are files in the repo, so they travel through git — one developer's fix becomes the whole team's default.

under the hood

spec — Definition of Ready

Parses and validates the spec contract fail-closed: schema, file paths, the AC⇄files⇄tests triple. An incomplete spec does not pass.

verify — quality gates

Concurrent gates, a single merge point, config-first resolution. Writes the verification.md that gates delivery.

lessons — team memory

Typed lessons in .marvin/memory/: add at delivery and debugging, search at intake. Near-duplicate guarded.
